    Intro to Mechanics Design

    Unity Programming Module

    Intro to Mechanics design is a 2nd Year module i was enrolled on at Staffordshire University.
    For this module i had to create a 2D character controller which had forgiveness mechanics to help the player navigate the map, and create other mechanics that "feel good" using audio and visuals;
    In this module i achieved a First at 77%
    Below is only a slice of my IMD work, the rest can be viewed on the IMD playlist which can be found on my YouTube

    Character Movement Designs

    Here are some of the designs i had to create diagrams for in the module to help explain the movement on the assessment forum

    Coyote Time - Being able to jump just after walking off the ledge, this prevents the need for perfect inputs of jumping from ledges

    Corner Corrections - Correcting the players position if a small percentage (a few pixels) hit as the player is trying to navigate so the intended path is still taken

    Jump Buffer - Pressing jump before pressing space will store the input, if the input is stored when landing on the ground the player shall jump again, this makes more fluid controls.

    Apex Zone is the section at the apex of the jump in which the player has an increased control over the player's air movement to allow for more precision jumps, this becomes useful when playing precision platformers such as Celeste or Super Meatboy
    Allowing the player to make precise jumps in platformers that don't require exact precision such as the Mario games helps prevent less skilled players from getting frustrated that they couldn't make jumps
