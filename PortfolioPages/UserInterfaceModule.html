    UI Design & Implementation

    Unreal Engine UI Module

    This page is dedicated to my work for the UI Design and Implementation module at Staffordshire University, in this module students were tasked with researching, designing and programming RPG Genre UI Elements. The criteria set for students were to create an established style of their own and create the following:

  • Pause Menus
  • HUD Elements
  • Shops
  • An Inventory
  • Other UI Elements the student wants to include

    • I decided that my theming would be more of a simplistic design due to my lack of 2D art skills, this led me to use more of a rounded design to provide a sleeker look to my UI. In addition to the elements we were suggested to create, i made a journal page to keep track of quests, crafting page for craftables and pause toggles. This module was difficult running alongside the Junior Collaborative project due to time being split between both modules.
    In this module i achieved a First at 76% marks
