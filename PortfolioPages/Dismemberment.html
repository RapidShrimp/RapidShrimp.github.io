    Skeletal Dismemberment Plugin

    Free Unreal Engine Plugin

    I created this C++ plugin when prototyping mechanic ideas for a game.
    This took a few weeks of prototyping to make and has no support.
    The plugin consists of...

  • Limb Severance
  • Limb Tethering
  • Socket Particle Effects
  • Animation State Handling
  • Editor Data asset Tool

    • Project is free to download and use, avaliable on GitHub, this plugin is currently working in 5.2.1
