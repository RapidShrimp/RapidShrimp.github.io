    Skeletal Dismemberment Plugin

    Free Unreal Engine Plugin

    I created this C++ plugin when prototyping mechanic ideas for a game.
    This took a few weeks of prototyping to make and is no longer supported, The plugin consists of...

  • Limb Severance
  • Limb Tethering
  • Socket Particle Effects
  • Animation State Handling
  • Editor Data asset Tool

    • Download is free Via GitHub, this plugin is currently working in 5.2.1 but im confident it will package into other versions of Unreal.
