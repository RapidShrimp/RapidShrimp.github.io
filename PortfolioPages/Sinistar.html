    Sinistar C++ Remake

    Low Level Gameplay Programming

  • Team Size: Solo Project
  • Role: Programmer
  • Duration: 8 Weeks
  • Grade Achieved: 70% - First Class

    • In the Low Level Gameplay Programming module at The University of Staffordshire, my task was to faithfully recreate a 1980's Williams Arcade Unit game from a small selection of games and improve upon the original.
    My Selection was the Game Sinistar. In creating this game I ended up making a small Game Engine using Object Oriented Programming to assist in the creation of this.
